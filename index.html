Este guia fornece informações essenciais sobre o gerenciamento do Firebase para o projeto de Memoriais e Ofícios da Secretaria de Saúde.

Acessar Console Firebase

Monitorar Métricas & Cotas Diárias

📊 Acessar o Dashboard de Uso

  1. Faça login no Console Firebase
  2. Clique em Ir para o console
  3. Selecione o projeto "MemoGestao" na lista
  4. No menu lateral, clique em Firestore
  5. No centro da tela terá um menu com várias opções, clique em Uso
  6. Visualize as métricas em tempo real

🔍 Métricas Principais a Monitorar

Operações de Leitura do Firestore

Número de documentos lidos diariamente. Limite padrão: 50.000/dia

Operações de Escrita do Firestore

Número de documentos modificados. Limite padrão: 20.000/dia

Operações de Exclusão

Número de documentos deletados. Limite padrão: 20.000/dia

Armazenamento do Banco de Dados

Espaço utilizado em GB. Limite gratuito: 1 GB

⚠️ Atingir o Limite Diário

O que acontece? A aplicação retorna erro 429 (Too Many Requests) e usuários não conseguem executar operações.

Como Resolver

  1. Aguardar Reset — As cotas resetam automaticamente à meia-noite (horário UTC)
  2. Contatar Suporte — Se necessário, abrir ticket com Google Cloud Support

Verificar Limites Atuais

No Console Firebase, acesse:

Configurações (⚙️) → Utilização e Faturamento

💳 Plano Blaze - Como Funciona a Cobrança

Bom saber: Nosso projeto utiliza o plano Blaze (Pay-as-you-go), que cobra apenas pelo que você usa, sem limites diários fixos.

📊 Estrutura de Preços do Firestore

Operações de Leitura

$0,06 por 100.000 operações

Cada documento consultado = 1 operação

Operações de Escrita

$0,18 por 100.000 operações

Cada documento criado ou atualizado = 1 operação

Operações de Exclusão

$0,02 por 100.000 operações

Cada documento deletado = 1 operação

Armazenamento

$0,18 por GB/mês

Os primeiros 1 GB são gratuitos

💰 Exemplo de Custo Mensal

Consumo típico do projeto:

  • 1 milhão de leituras/mês: 1.000.000 ÷ 100.000 × $0,06 = $0,60
  • 500 mil escritas/mês: 500.000 ÷ 100.000 × $0,18 = $0,90
  • 100 GB de armazenamento: (100 - 1) × $0,18 = $17,82
  • Total estimado: ~$19,32/mês

📱 Monitorar Gastos em Tempo Real

  1. Acesse o Google Cloud Console
  2. Clique em Faturamento no menu lateral
  3. Selecione o projeto "MemoGestao"
  4. Veja o gasto do mês atual em tempo real
  5. Clique em Relatórios para ver histórico detalhado

⚠️ Como Receber Alertas de Custo

  1. No Google Cloud Console, vá em Faturamento → Orçamentos e alertas
  2. Clique em Criar Orçamento
  3. Defina um valor limite (ex: $50/mês)
  4. Configure alertas em: 50%, 90% e 100% do orçamento
  5. Adicione emails para receber notificações automáticas
Dica importante: Configure alertas para não ser surpreendido! Recomendamos um limite de $50-100/mês.

✅ Verificar se está Rodando Normalmente

Health Check

  1. Acesse a aplicação: memoriais-e-oficios.vercel.app
  2. Tente criar um novo memorial ou ofício (teste de escrita)
  3. Tente acessar a lista de documentos (teste de leitura)
  4. Se ambos funcionarem, o banco está operacional ✓

Verificação Técnica no Console

  1. Vá em Firestore
  2. Na seção Dados
  3. Verifique se há dados nas coleções esperadas
Tudo OK? Se consegue ler e escrever dados sem erros, o serviço está funcionando normalmente.
